sort
배열 정렬 가능
오름차순 정렬
기존 배열을 변형시킴 
=> 이게 매우 중요해 그래서  콘솔을 찍을때, 원래 배열 변수명을 사용한다.

// 오름차순
numbers.sort((a, b) => {
	return a - b 
});

// 내림차순
numbers.sort((a, b) => {
	return b - a
});
